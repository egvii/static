```callout
type: success
title: What are Conditional Sentences?
Conditional sentences express a condition and its result. They have two parts:
- **If-clause** — the part expressing the condition
- **Main clause** — the part expressing the result

## Be / Have / Do — Verb Forms

::grid
==red:Be की form==
`green:be, am, are`[v1] · `blue:was, were`[v2] · `red:been`[v3] · being · `green:is`[v5]
---
==red:Have की form==
`green:have`[v1] · `blue:had`[v2] · `red:had`[v3] · having · `green:has`[v5]
---
==red:Do की form==
`green:do`[v1] · `blue:did`[v2] · `red:done`[v3] · doing · `green:does`[v5]
::
```

```timeline maroon
Be | `green:be, am, are`[v1] · `blue:was, were`[v2] · `red:been`[v3] · being · `green:is`[v5]
```

```section
color: green
number: 1
title: "First Conditional — Real Future Possibility"
```

**Structure:** If + S + V1/V5…, S + will + base form + ……

Used when the condition is **likely to happen** in the future (Open Conditionals).

```cards
### State — होना (teal)
- If the manager **is** unavailable, we **will** reschedule the meeting.
- _अगर मैनेजर उपलब्ध नहीं होंगे, तो हम मीटिंग का समय बदलेंगे।_
- If the system **is** down, the portal **will** not work.
- _अगर सिस्टम डाउन होगा, तो पोर्टल काम नहीं करेगा।_
---
### Possession — पास होना (blue)
- If she **has** the project files, she **will** share them by evening.
- _अगर उसके पास प्रोजेक्ट फाइल्स हैं, तो वह शाम तक भेज देगी।_
- If we **have** your consent, we **will** proceed with the plan.
- _अगर हमें आपकी अनुमति मिलती है, तो हम योजना आगे बढ़ाएंगे।_
---
### Action — काम होना (orange)
- If the client **responds** today, we **will** finalize the budget.
- _अगर क्लाइंट आज जवाब देता है, तो हम बजट फाइनल कर देंगे।_
- If I **complete** the report, I **will** mail it by 5 PM.
- _अगर मैंने रिपोर्ट पूरी की, तो मैं 5 बजे तक मेल कर दूंगा।_
```

```section
color: navy
number: 2
title: "Second Conditional — Unreal / Imaginary"
```

**Structure:** If + S + V2…, S + would + base form + ……

Used when the condition is **not true or unlikely** (Semi-Closed).

```cards
### State — होना (teal)
- ==red:Rule== — Use only **were** for all subjects
- If I `red:was[were]` the CEO, I **would** introduce a 4-day workweek.
- _अगर मैं CEO होता, तो मैं हफ्ते में 4 दिन की नौकरी लागू करता।_
- If she `red:was[were]` more disciplined, she **would** achieve more.
- _अगर वह ज्यादा अनुशासित होती, तो ज्यादा हासिल करती।_
---
### Possession — पास होना (blue)
- ==red:Note== — Has/Have/Had → अगर इनके बाद V3 नहीं आया, तो ये possession show कर रहे
- If I **had** a degree in law, I **would** open my own firm.
- _अगर मेरी लॉ में डिग्री होती, तो मैं अपनी फर्म खोलता।_
- If he **had** more savings, he **would** invest in a start-up.
- _अगर उसके पास ज्यादा सेविंग्स होती, तो वह स्टार्टअप में निवेश करता।_
---
### Action — काम होना (orange)
- If I **worked** in Mumbai, I **would** travel by metro.
- _अगर मैं मुंबई में काम करता, तो मेट्रो से जाता।_
- If we **knew** the route, we **would** reach early.
- _अगर हमें रास्ता पता होता, तो हम जल्दी पहुँचते।_
```

```section
color: maroon
number: 3
title: "Third Conditional — Past Regret"
```

**Structure:** If + S + had + V3…, S + would have + V3……

Used to express **past regrets or missed opportunities** (Closed Condition).

```cards
### State — होना (teal)
- ==red:Rule== — Use "had been" (been = V3 form of be)
- If he **had been** attentive, he **wouldn't have** missed the details.
- _अगर वह ध्यान देता, तो विवरण नहीं छूटते।_
- If I **had been** patient, things **would have** turned out better.
- _अगर मैं धैर्य रखता, तो सब बेहतर होता।_
---
### Possession — पास होना (blue)
- ==red:Rule== — Had (helping verb) + Had (V3)
- If she **had had** her ID, she **would have been** allowed in.
- _अगर उसके पास ID होती, तो उसे अंदर जाने देते।_
- If we **had had** more time, we **would have** explored the museum.
- _अगर हमारे पास और समय होता, तो हम म्यूजियम देखते।_
---
### Action — काम होना (orange)
- If I **had submitted** the form earlier, I **would have** secured the seat.
- _अगर मैंने फॉर्म पहले जमा किया होता, तो सीट मिल जाती।_
- If she **had prepared** well, she **would have** cracked the interview.
- _अगर वह अच्छे से तैयार होती, तो इंटरव्यू निकाल लेती।_
```

```highlight

title: The Core Rules

::note
If के साथ कभी `red:will[]` नहीं आता
---
1st vs 2nd — दोनों present & future situation की बात करते हैं, फ़र्क सिर्फ़ Reality vs Imagination है
---
2nd vs 3rd — 2nd = present में कल्पना (If I had time now…), 3rd = past का पछतावा (If I had had time last year…)
::
```

```numbered purple
1 | Real Possibility (V1)
If I **work** hard, I **will** pass. → V1 → Real possibility
---
2 | Imagination (V2)
If I **worked** hard, I **would** pass. → V2 → Only imagination
---
3 | Past Regret (Had + V3)
If I **had worked** hard, I **would have** passed. → Had + V3 → Past regret
```
